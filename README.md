## 📝 Sobre o Projeto

O **TrucoScoreBoard** nasceu da necessidade de uma ferramenta simples, porém robusta, para gerenciar a pontuação de partidas de Truco (seja Paulista ou Mineiro). O foco principal é a **usabilidade**, garantindo que os jogadores não percam tempo configurando o app enquanto o jogo está pegando fogo.

### ✨ Funcionalidades Principais

* **Controle de Pontos:** Adição e subtração rápida de tentos.
* **Botão de Reset:** Inicie uma nova partida com apenas um clique.

## 🛠️ Como Executar o Projeto

Para rodar este projeto localmente, você precisará do **Android Studio** instalado.

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/fereziniNi/TrucoScoreBoard.git](https://github.com/fereziniNi/TrucoScoreBoard.git)
    ```
2.  **Abra o projeto:**
    Inicie o Android Studio e selecione `Open an existing project`.
3.  **Sincronize o Gradle:**
    Aguarde o download das dependências.
4.  **Execute:**
    Conecte um dispositivo físico ou use um emulador e clique no botão `Run`.
